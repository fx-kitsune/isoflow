# Cài đặt

Isoflow được phát hành dưới dạng một **React component** mà bạn có thể nhúng vào dự án của mình.

Để cài đặt bằng `npm`:

```bash
npm install isoflow
```

hoặc `yarn`:

```bash
yarn add isoflow
```

### Bản thử nghiệm

Phiên bản mới nhất của Isoflow luôn được đồng bộ hóa tại đây trên [CodeSandbox](https://codesandbox.io/p/sandbox/github/markmanx/isoflow).

### Chạy Isoflow ở chế độ phát triển
Để chạy Isoflow trên máy cục bộ của bạn:

1. Sao chép [Kho lưu trữ Github](https://github.com/markmanx/isoflow).
2. `npm i`
3. `npm run start`.

### Tài liệu cho nhà phát triển
Để xem tài liệu API chi tiết, các ví dụ và nhiều hơn nữa, hãy tham khảo [tài liệu dành cho nhà phát triển](https://v2.isoflow.io/docs) trực tuyến. Bạn cũng có thể biên dịch và chạy tài liệu cục bộ:

- `npm run docs:build`
- `npm run docs:start`
